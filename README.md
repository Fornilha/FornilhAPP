# FornilhApp

App de gestão interna do Fornilha — controle de desempenho, cardápio, colaboradores e faturamento.

## Como rodar

Abra o arquivo `index.html` direto no navegador. Nenhuma instalação necessária.

## Estrutura

```
FornilhAPP/
├── index.html          # App completo (single-page)
└── Dados do Fornilha/  # PDFs e dados do negócio
```
